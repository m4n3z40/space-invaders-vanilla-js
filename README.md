space-invaders-vanilla-js
=========================

Space Invaders Game written in vanilla javascript.


--------------------------------------------------

### Technologies & libs used ###

 * Gulp
 * Jade
 * Browserify
 * Stylus
 * lodash
 * Javascript (S2)

##### Instructions for developing ####

 * Clone this repository
 * Run "npm install"
 * Run "gulp serve"

##### Instructions for building ####

 * Clone this repository
 * Run "npm install"
 * Run "gulp"
